# toko-buah-Asyifa Alya Nabila
Aplikasi Toko Buah 

**APLIKASI TOKO BUAH**
Aplikasi Toko Buah adalah aplikasi berbasis web yang digunakan untuk mengelola data buah seperti nama buah, harga, dan stok. Aplikasi ini dibuat menggunakan bahasa pemrograman PHP, database MySQL, dan framework Bootstrap untuk tampilan.
